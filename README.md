# PatternMatchingTool

> WPF 기반의 이미지 패턴 매칭 툴

## 프로젝트 설명

PatternMatchingTool은 이미지에서 특정 패턴을 찾고, 결과 영역을 표시하며, OCR 및 바코드 정보까지 관리할 수 있는 WPF 기반 도구입니다.  
- 템플릿 매칭 기능  
- 특징점 기반 매칭 (SIFT)  
- Rectangle 및 Corner Ellipse를 통한 GUI 패턴 편집  
- OCR Predict 정보 및 바코드 정보 관리  
- TCP/IP 통신 기능으로 외부 시스템과 연동 가능  

---

## 기능

- 이미지 불러오기 / 카메라 Grab  
- 패턴 등록 및 편집 (Rectangle + 모서리 핸들)  
- 결과 Rect 표시  
- 특징점 추출 및 저장/로드  
- OCR Predict 정보 업데이트  
- 바코드 리딩 기능  
- TCP/IP 통신을 통한 외부 장치 데이터 전송/수신
